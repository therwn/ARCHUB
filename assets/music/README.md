# Müzik Dosyaları

Bu klasöre 3 adet MP3 dosyası ekleyin:

1. `arc1.mp3`
2. `arc2.mp3`
3. `arc3.mp3`

Site açıldığında bu şarkılardan biri rastgele seçilir ve çalmaya başlar. Şarkı bittiğinde diğer şarkılardan biri otomatik olarak çalmaya devam eder.

**Not:** Dosya isimleri `script.js` dosyasındaki `musicFiles` dizisiyle eşleşmelidir.
